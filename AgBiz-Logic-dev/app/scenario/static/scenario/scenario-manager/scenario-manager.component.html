AgBiz Logic Scenarios

Scenario Manager

AgBiz Logic tools rely on Scenarios to run decision-support analyses. You can select an existing Scenario and modify it according to your current objectives, copy an existing Scenario and edit the copy as needed or create a new Scenario by clicking on the Create New Scenario button.

AgBizProfit

Plan for the future... Today

AgBizProfit relies on Scenarios to run a profitability of investments analysis. You can either select an existing Scenario and modify according to your current objectives, or you can create a new Scenario by clicking on the Create New button.

AgBizFinance

AgBizFinance allows you to create your own Scenarios or use existing Scenarios from AgBizProfit and AgBizLease as the foundation for annual returns and cash costs for each business enterprise. With current financial information, distributions, depreciation, inflation, and future transactions for each AgBizFinance Scenario, you can compare Cash Flow, Balance Sheets, and Income Statements.
